## 已支持功能:

- 样式美化 | 自定义背景
- 视频页样式微调
- 视频快捷分享复制模式
- 移除评论关键字搜索图标
- 移除评论关键字搜索跳转
- 评论区过滤(自定义关键词未完成)
- 页面宽屏
- 护眼蒙版
- 快捷键增强
