## 已支持功能:

- 免登录复制
  - ![免登录复制效果图](https://s2.loli.net/2023/10/22/ulQ75mbK1TSFOxi.png)
- 样式美化|沉浸式阅读
  - 调整页面宽度
- 移除顶部导航或不跟随
- 隐藏底部工具栏或不跟随
- 去广告
- UI 调整
  - 每一个卡片都可以关闭
- 调整字体大小
- 自定义背景
- 专注模式
  - 快捷键 Ctrl+Enter 默认关闭
